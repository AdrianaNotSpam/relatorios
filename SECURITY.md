# Segurança

Este repositório aloja relatórios HTML estáticos do estúdio notspam.studio, publicados via GitHub Pages.

## Reportar um problema

Se encontrares uma vulnerabilidade ou exposição inesperada de conteúdo:

- Email: ola@notspamstudio.com
- Em alternativa, usa o GitHub *Private Vulnerability Reporting* (separador *Security* deste repo)

Por favor não abras issue público para problemas de segurança.

## O que esperamos

- Resposta inicial em 3 dias úteis
- Resolução de fugas activas em 24h
- Coordenação para divulgação após o fix
